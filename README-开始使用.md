## 🎯 快速开始（5分钟上手）

### 第一步：创建项目文件夹

```bash
# 在你喜欢的位置创建项目文件夹
mkdir fittrack
cd fittrack
```

### 第二步：放置配置文件

将以下文件复制到`fittrack`文件夹：
- ✅ NOTES.md
- ✅ .cursorrules
- ✅ 创建`augment`子文件夹，将PRD和技术设计文档放入

### 第三步：打开Cursor

1. 启动Cursor应用
2. 点击"Open Folder"
3. 选择`fittrack`文件夹
4. Cursor会自动检测到`.cursorrules`文件

### 第四步：开始对话

在Cursor中按 `Ctrl+L`（或 `Cmd+L` on Mac）打开AI助手，输入：

```
请先阅读NOTES.md文件，然后帮我搭建FitTrack项目的基础环境。
```

**AI会自动：**
- ✅ 读取NOTES.md了解项目需求
- ✅ 按照.cursorrules的规则工作
- ✅ 用简单的语言解释每一步
- ✅ 生成可直接运行的代码

## 🐛 常见问题解决

### Q1: Cursor没有检测到.cursorrules？

**解决方案：**
1. 确保文件名是`.cursorrules`（前面有点）
2. 重启Cursor
3. 重新打开项目文件夹

### Q2: AI不理解我的意图？

**解决方案：**
```
你：请先读取NOTES.md文件，了解FitTrack项目的完整背景
```

### Q3: 代码报错怎么办？

**解决方案：**
```
你：我遇到这个错误：
[完整的错误信息]

请帮我解决，并用简单的语言解释错误原因
```

### Q4: 想改动AI生成的代码？

**解决方案：**
```
你：我想把按钮颜色改成绿色，怎么改？

你：能不能把这个功能简化一下？
```
