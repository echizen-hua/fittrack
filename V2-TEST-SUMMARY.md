**文件位置：** `lib/exerciseVideos.ts` + `app/add/page.tsx`

- ✅ 点击链接在新标签页打开
- ✅ 链接文本友好显示（"📹 查看[动作名]视频教程"）

**支持的动作：**
- 腿部：深蹲、前蹲、硬拉、罗马尼亚硬拉、腿举、弓步蹲

- 背部：引体向上、高位下拉、杠铃划船、坐姿划船
- 肩部：推举、侧平举、面拉
- 手臂：二头弯举、三头下压、锤式弯举

## 🧪 测试指南

### 快速测试步骤

#### 1. 测试身体数据追踪
```
1. 访问 http://localhost:3000/body
2. 填写：体重 70.5kg，体脂率 15%，肌肉量 55kg
3. 点击保存
4. 检查历史记录是否显示
```

#### 2. 测试训练计划
```
1. 访问 http://localhost:3000/plans
2. 点击"新手全身训练"
3. 查看计划详情
4. 点击"开始记录训练"按钮
```

#### 3. 测试分享功能
```
1. 访问 http://localhost:3000/history
2. 找到一条记录
3. 点击"📤 分享"按钮
4. 检查剪贴板内容
```

#### 4. 测试视频教程
```
1. 访问 http://localhost:3000/add
2. 选择"深蹲"
3. 检查是否显示视频教程链接
4. 点击链接测试
```
