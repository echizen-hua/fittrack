## 📊 数据库设计（极简版）

### 数据表结构

#### 1. Users表（用户）
```sql
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);
```

#### 2. Workouts表（训练记录）
```sql
CREATE TABLE workouts (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  exercise_name VARCHAR(100) NOT NULL,  -- 动作名称（如"深蹲"）
  weight DECIMAL(5,2) NOT NULL,         -- 重量（kg）
  reps INTEGER NOT NULL,                -- 次数
  sets INTEGER NOT NULL,                -- 组数
  created_at TIMESTAMP DEFAULT NOW()
);

-- 索引（提升查询速度）
CREATE INDEX idx_workouts_user_id ON workouts(user_id);
CREATE INDEX idx_workouts_created_at ON workouts(created_at DESC);
```

#### 3. Exercises表（预设动作库）
```sql
CREATE TABLE exercises (
  id UUID PRIMARY KEY,
  name VARCHAR(100) NOT NULL,           -- 深蹲
  category VARCHAR(50) NOT NULL,        -- 腿部
  english_name VARCHAR(100)             -- Squat
);
```

### 关键概念速查

#### 什么是Next.js？
> 一个React框架，帮你快速搭建网站，不用从零配置。

**为什么用它**：
- ✅ 自带路由（文件夹=页面）
- ✅ 自动优化（网站快）
- ✅ AI很熟悉（容易生成代码）

---

#### 什么是Supabase？
> 一个后端服务，提供数据库、认证、存储，不用自己搭服务器。

**核心功能**：
- 数据库（PostgreSQL）
- 用户认证（登录注册）
- 实时订阅（数据变化自动更新）

**为什么用它**：
- ✅ 免费额度够MVP用
- ✅ 不用写后端代码
- ✅ 文档友好，AI能理解

---

#### 什么是PWA？
> Progressive Web App，网页应用但像原生APP。

**特点**：
- 可以添加到主屏幕
- 可以离线使用（V2功能）
- 不需要下载安装

**FitTrack的PWA配置**（Cursor自动生成）：
- manifest.json（应用图标、名称）
- service-worker.js（离线功能，可选）
